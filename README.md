# 4-Digit Brute Force Generator 

A beginner-friendly Python script that generates all possible 4-digit combinations (0000 to 9999).

This project focuses on learning nested loops and basic brute-force concepts.


# Features

Generates every 4-digit combination

Uses simple nested loops

Beginner-friendly Python code

# Requirements
Python 3.x

# Usage
Run the script:

python bruteforce_4digits.py
